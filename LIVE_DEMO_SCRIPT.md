# 🎤 AI MBA 발표용 실시간 코딩 데모 스크립트

## 🎯 데모 목표
**"코드를 저장하면 즉시 화면에 반영됩니다"**를 실시간으로 증명

---

## 📋 준비 단계 (발표 전)

### 1. 개발 모드 실행
```bash
./dev.sh
# 또는
npm run dev
```

### 2. 확인 사항
- ✅ 개발자 도구가 자동으로 열렸는지
- ✅ 콘솔에 "🔥 Hot reload enabled" 메시지
- ✅ 앱이 정상 실행되는지

### 3. 에디터 준비
- VS Code에서 `styles.css`, `index.html`, `renderer.js` 열어두기
- 화면 분할: 왼쪽 에디터, 오른쪽 GOYO 앱

---

## 🎬 데모 시나리오 (5분)

### 데모 1: CSS 색상 즉시 변경 (30초)

**말하기:**
> "먼저 대시보드의 색상을 실시간으로 변경해보겠습니다."

**실행:**
1. `styles.css` 파일 열기 (8번 줄 근처)
2. `.stat-card` 찾기
3. 색상 변경:

```css
/* 변경 전 */
.stat-card {
  background: linear-gradient(135deg, #1a1c2e 0%, #2b2d42 100%);
}

/* 변경 후 */
.stat-card {
  background: linear-gradient(135deg, #ff6b6b 0%, #ee5a6f 100%);
}
```

4. **`Ctrl+S` (저장)**
5. 👉 **즉시 화면에 반영됨!**

**강조:**
> "보시다시피, 저장하자마자 앱을 재시작하지 않고도 색상이 바로 변경되었습니다."

---

### 데모 2: 텍스트 실시간 수정 (30초)

**말하기:**
> "이번에는 대시보드 제목을 실시간으로 바꿔보겠습니다."

**실행:**
1. `index.html` 파일 열기
2. 대시보드 제목 찾기 (약 60번 줄)
3. 텍스트 수정:

```html
<!-- 변경 전 -->
<h2>실시간 감정 모니터링 대시보드</h2>

<!-- 변경 후 -->
<h2>🔥 AI MBA 실시간 데모 - 감정 모니터링 🔥</h2>
```

4. **`Ctrl+S` (저장)**
5. 👉 **즉시 화면에 반영됨!**

**강조:**
> "HTML 구조도 저장과 동시에 실시간으로 업데이트됩니다."

---

### 데모 3: 새로운 기능 추가 (1분)

**말하기:**
> "이제 새로운 알림 기능을 실시간으로 추가해보겠습니다."

**실행:**
1. `renderer.js` 파일 열기 (맨 아래로 스크롤)
2. 새 함수 추가:

```javascript
// 실시간 데모 함수 추가
function showLiveDemo() {
  const currentTime = new Date().toLocaleTimeString('ko-KR');
  showNotification(`🎤 실시간 코딩 데모! (${currentTime})`, 'success');
}

// 5초마다 실행
setInterval(showLiveDemo, 5000);
```

3. **`Ctrl+S` (저장)**
4. 👉 **5초 후 자동으로 알림이 나타남!**

**강조:**
> "이렇게 새로운 기능을 추가해도 앱을 재시작할 필요가 없습니다. 개발 속도가 획기적으로 빨라지죠."

---

### 데모 4: 모바일 체크인 버튼 스타일 변경 (30초)

**말하기:**
> "마지막으로 모바일 체크인 버튼의 스타일을 바꿔보겠습니다."

**실행:**
1. "데이터 수집" 메뉴로 이동
2. "모바일 체크인" 탭 클릭
3. `styles.css`에서 `.mobile-submit-btn` 찾기
4. 스타일 변경:

```css
/* 변경 전 */
.mobile-submit-btn {
  background: linear-gradient(135deg, #06d6a0 0%, #118ab2 100%);
}

/* 변경 후 */
.mobile-submit-btn {
  background: linear-gradient(135deg, #ffd60a 0%, #ffc300 100%);
  box-shadow: 0 8px 30px rgba(255, 214, 10, 0.4);
  animation: pulse-button 2s infinite;
}

@keyframes pulse-button {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.05); }
}
```

5. **`Ctrl+S` (저장)**
6. 👉 **버튼이 노란색으로 변하고 펄스 애니메이션!**

**강조:**
> "이처럼 Electron 앱도 웹 개발처럼 빠르게 개발하고 즉시 확인할 수 있습니다."

## 🎯 핵심 메시지 전달

1. **개발 생산성**

2. **실시간 피드백**

4. **디버깅 편의성**

## ⚡ 빠른 복구 (문제 발생 시)

### 데모 중 문제가 생기면

**Hot Reload가 작동하지 않을 때:**
```bash
# 앱 종료 (Ctrl+C)
# 다시 시작
npm run dev
```

**수동 새로고침:**
- `Ctrl+R` (Windows/Linux)
- `Cmd+R` (Mac)

**안전 복구:**
```bash
# 모든 변경사항 되돌리기
git checkout -- styles.css index.html renderer.js
npm run dev
```
