## 💡 주요 기능 체험 가이드

### ⭐ 얼굴 인식 AI 체크인
1. `📝 데이터 수집` 탭 클릭
2. `얼굴 인식 AI` 탭 선택
3. `📷 카메라 시작` 버튼 클릭
4. 웹캠 권한 허용
5. 얼굴을 화면에 맞추기
6. `📸 사진 촬영` 버튼 클릭
7. 2초 후 AI 분석 결과 확인
8. 자동으로 체크인 폼 채워짐
9. `✅ 체크인 완료` 버튼 클릭

### ⭐ 웨어러블 센서 시뮬레이션
1. `📝 데이터 수집` 탭 클릭
2. `자동 수집 시뮬레이션` 탭 선택
3. `🚀 시뮬레이션 시작` 버튼 클릭
4. 실시간 센서 데이터 생성 (10초마다)
   - 심박수: 60-100 bpm
   - 스트레스 지수: 0-100
   - 수면 질: 50-100%
   - 활동량: 0-10,000 보
5. 스트레스 ≥70 → 자동 알림 생성
6. 타임라인에서 실시간 데이터 확인
7. `⏸️ 시뮬레이션 중지` 버튼으로 종료

### ⭐ 스크린샷으로 모든 화면 한눈에 보기
1. `👁️ 미리보기` 탭 클릭
2. `📸 스크린샷` 탭 선택
3. 8개 주요 화면 카드 확인
4. 각 카드 호버 시 확대
5. `🔍 크게 보기` 버튼 클릭 → 풀스크린 모달
6. ESC 또는 배경 클릭으로 닫기

## 🔍 Chrome DevTools 활용

### Console에서 API 직접 호출
```javascript
// 브라우저 콘솔 (F12) 열기
// 직원 목록 조회
const workers = await window.api.invoke('get-active-workers');
console.table(workers);

// 감정 로그 조회
const logs = await window.api.invoke('get-emotion-logs', {
  startDate: '2026-01-01',
  endDate: '2026-02-18'
});
console.log('Total logs:', logs.length);

// 대시보드 통계 조회
const stats = await window.api.invoke('get-dashboard-stats');
console.log('Dashboard stats:', stats);
```

### Network 탭에서 API 확인
1. F12 → Network 탭
2. 탭을 클릭하여 이동
3. `POST /api/` 요청 확인
4. Request Payload: `{ channel: "get-active-workers", data: null }`
5. Response: JSON 데이터 확인
