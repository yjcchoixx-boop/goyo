## 🐛 문제 발생 시 체크리스트

### 탭이 활성화되지 않는 경우:

1. **브라우저 새로고침** (Ctrl+R 또는 F5)
2. **개발자 도구 확인** (F12)
   - Console 탭에서 JavaScript 오류 확인
   - Network 탭에서 파일 로딩 확인
3. **브라우저 캐시 강제 삭제** (Ctrl+Shift+R)

### 데이터가 표시되지 않는 경우:

1. **API 테스트**
   ```bash
   curl https://3000-i3mj7uf0pn6q669k7it2m-0e616f0a.sandbox.novita.ai/api/statistics
   ```
2. **개발자 도구 Network 탭** 확인
3. **서버 로그 확인**

### 실시간 업데이트가 안 되는 경우:

1. **일시정지 상태 확인** (일시정지 버튼이 "재개"로 표시되어 있는지)
2. **브라우저 새로고침**
3. **수동 새로고침 버튼** 클릭
